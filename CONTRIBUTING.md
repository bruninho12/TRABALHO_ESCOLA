# Contribuindo com o DespFinancee

Obrigado por considerar contribuir com o projeto!

Este arquivo é um esqueleto inicial para documentação de contribuição.  
Sugestão de próximos detalhes a adicionar:

- Fluxo de desenvolvimento (branching, commits e PRs)
- Padrões de código (lint, formatação, testes)
- Como rodar o backend e frontend em desenvolvimento
- Como abrir issues bem descritas

Enquanto isso, você pode:

1. Abrir uma issue descrevendo a mudança desejada.
2. Criar um fork do repositório.
3. Abrir um Pull Request com uma descrição clara do que foi alterado.
