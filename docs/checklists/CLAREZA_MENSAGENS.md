# Clareza de erros e mensagens

- Mensagens de erro são exibidas em português claro
- Feedback visual para sucesso, erro e validação
- Toasts, modais ou alertas para ações importantes
- Mensagens de validação detalham o campo e o motivo
- Erros de autenticação e conexão são amigáveis

## Checklist

- [ ] Mensagens de erro são compreensíveis
- [ ] Mensagens de sucesso são destacadas
- [ ] Validações mostram campo e motivo
- [ ] Erros de autenticação são amigáveis
- [ ] Erros de conexão orientam o usuário
